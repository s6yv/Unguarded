Unguarded is a short 2D platformer created in high school as a Unity and game-design learning project during the 2020-2021 academic year.

The repository contains a playable Windows build. All visual assets and music used in the project were created by me.

## Running the game

1. Download the repository by selecting **Code > Download ZIP** on GitHub.
2. Extract the entire ZIP archive.
3. Run `CS IA.exe`.

Keep `CS IA.exe`, `CS IA_Data`, and `MonoBleedingEdge` together in the same directory. The executable depends on the files in those folders.

## Controls

| Input | Action |
| --- | --- |
| Left mouse button | Create a ball that damages enemies |
| `E` | Create a damaging area |

## Project goals

This project was created to practice:

- Building and packaging a game with Unity
- Basic 2D platformer and combat design
- Implementing player interactions and enemy damage
- Creating and integrating original visual assets
- Producing and adding original music

## Technology

- Unity
- C#
- Windows standalone build

## Repository contents

This repository contains the compiled game rather than the Unity source project.

- `CS IA.exe` - game executable
- `CS IA_Data/` - game data and managed assemblies
- `MonoBleedingEdge/` - Unity runtime files
- `UnityPlayer.dll` - Unity player library
